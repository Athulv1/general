Flooring Layout — Config Builder

Enter your DXF layer names. Only the store outline is required — everything else is optional.

1. Store outline REQUIRED

The layer that holds one closed shape around the whole floor area to be tiled.

Must be a closed polyline/polygon. An open line will not work.
⚠ Store outline layer is required.

2. No-floor areas OPTIONAL

Closed shapes around rooms to leave empty (toilet, store room, staff area). Add one row per layer.

3. Walls OPTIONAL

Wall / wall-hatch layers to copy into the output for context. Toggle off to omit walls.

4. Tile start point OPTIONAL

A layer with one POINT or block marking where tiling begins (usually the door). Leave blank to auto-detect.

5. Tile settings

How the tiles are laid and ordered.

6. Files OPTIONAL

Input DXF and output paths. Can also be passed on the command line instead.
