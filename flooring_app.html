Flooring + Speaker Layout Generator

Upload a DXF, define zones once, then assign flooring and speakers to them.

STEP 1

Upload your DXF

We read its layers, blocks and text labels so you can pick them below.

⬆ Click or drag a .dxf here
STEP 2

Store outline REQUIRED

Layer holding one closed shape around the whole floor.

STEP 3

Zones SHARED

Define every region once here — a closed polyline, a block, or a room found by its text label (balloon). Flooring and Speakers below both reference these by name (no duplication).

STEP 4

Flooring types

Define each finish once: a tile (length×width×rotation) or a hatch pattern. Assign them to zones below.

STEP 5

Flooring per zone

Give a zone a flooring type. Leave this empty to floor the whole store with the default type below.

STEP 6

Speaker layout OPTIONAL

Place speakers in a zone, a named room, the front-of-house, or across the whole store. Each speaker covers a circle (its coverage radius) — leave a count blank to auto-fit speakers to the area.

Coverage radius = how far one speaker reaches (the "balloon").
Speakers are auto-placed on a grid so every point is within the coverage radius. Count is computed automatically.
Box dims are for the amplifier box placed below.
STEP 7

Sprinkler layout OPTIONAL

Fills the store with sprinkler heads on a grid so every point is within a head's coverage radius — 100% coverage, no gaps. Count is computed automatically.

Coverage radius = the circle each head protects. For 100% cover the grid spacing is min(radius×√2, max spacing). Max spacing caps center-to-center distance (fire code).
STEP 8

Partition / marking plan OPTIONAL

Produces a clean marking plan: keeps only the layers you list and removes everything else, colours your zone outlines, and can label each zone's area. The store outline is always kept.

Layers to keep — everything else is removed. A furniture/fixture block is kept only if it sits on a Walls & structure layer.

STEP 9

Electrical layout OPTIONAL

Places electrical points — Wall (WP), Ceiling (CP), Floor (FP) and Data (CDP) — by matching fixture groups (block / layer name patterns) and rendering them relative to the zones below. The store outline is the boundary.

Point groups — match fixtures by layer / block name, choose where the point goes, and give its label + spec note.

STEP 10

Finishes & BOQ OPTIONAL

Marks wall finishes on the plan — paint, tile, mirror, fluting, laminate, ACP — and totals each one's area for the BOQ. Area = wall length × the height you give.

Finishes — name the material, where it goes (zone names from Step 3, or layer / block names), and its height up the wall.

STEP 11

Skirting layout OPTIONAL

Runs the skirting — the short strip where the floor meets the wall — around the rooms you choose. Measured in running metres; follows real walls (uses your Walls layers) and skips door openings.

Skirting runs along real walls only and skips door openings (set your door layers in the Walls section below). Zones in the exclude list get no skirting at all.

Walls

The output is clean — walls + flooring + speakers only (furniture, fixtures and dimensions are not copied). List the structural layers to copy into the output as the wall frame: walls, partitions, columns, beams, glazing.

Rescales the hatch pattern inside wall layers when copied to the output.
